# Sales-Forecasting-Time-Series-

## Project Overview
This project analyzes monthly sales data and forecasts future sales using ARIMA time series modeling.

## Objectives
- Understand time series data
- Identify trends and seasonality
- Forecast future sales
- Visualize predictions

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Statsmodels

## Workflow
1. Data loading and preprocessing
2. Trend visualization
3. Time series decomposition
4. ARIMA model training
5. Forecast generation

## Results
The model successfully identifies sales trends and predicts future values to assist business planning.

- SARIMA model
